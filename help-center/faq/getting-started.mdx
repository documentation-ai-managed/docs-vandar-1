## Overview

To use Vandar, sign up and get your API key. All API requests require it.

## Steps

1. Go to `https://panel.vandar.io/register`
2. Complete sign-up and verification
3. In the dashboard, open **API Keys**
4. Click **Create API Key**
5. Copy and store securely

## Example Usage

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://api.vandar.io/v2/me
```
