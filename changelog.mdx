<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added support for bulk product uploads with CSV import for faster listing management.
- Introduced automated delivery notifications for digital game accounts.

## Improvements

- Optimized product search to handle Arabic and English queries seamlessly.
- Enhanced checkout flow with one-click purchase for returning customers.

## Bug Fixes

- Fixed pricing display issues on high-traffic product pages.
- Resolved cart persistence problems during session timeouts.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["bugfix", "improvement"]}>

## Improvements

- Upgraded payment gateway integration for faster transaction processing.
- Improved mobile responsiveness for product galleries and bestseller sections.

## Bug Fixes

- Corrected inventory sync errors for shared accounts.
- Patched security vulnerability in user session management.

## Policy Changes

- Updated refund policy to include 48-hour window for digital delivery disputes.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## Breaking Changes

## Initial Setup

- Buyer accounts with order history tracking.

## How to Upgrade

<Tabs>
  <Tab title="Sellers" icon="package">

    ```bash
    # Update your seller dashboard
    npm update vandar-seller-sdk@latest
    ```
    Restart your integration and clear cache.
  </Tab>
  <Tab title="Buyers" icon="shopping-cart">
    No action required - updates deploy automatically.
  </Tab>
</Tabs>
